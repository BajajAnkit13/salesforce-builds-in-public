## Focus Areas

This repository emphasizes both **Salesforce platform skills** and **core software engineering principles**, including:

- **Object-Oriented Programming (OOP):**  
  Encapsulation, Abstraction, Inheritance, and Polymorphism are applied through Apex classes, interfaces, and handler frameworks

- **Apex Development:**  
  Triggers, trigger handler patterns, batch & async processing, exception handling, and bulkification

- **Lightning Web Components (LWC):**  
  Component-based UI design, reusability, and separation of concerns

- **SOQL & Performance:**  
  Governor-limit-safe queries, selective filters, and scalable data access patterns

- **Integrations:**  
  REST API callouts using Named Credentials, external services, and error handling

- **Test-Driven Development:**  
  Meaningful test classes with 90%+ coverage focusing on behaviour, not just lines of code
